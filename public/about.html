Our Professional Interests

As web developers, we are particularly focused on frontend development and user experience design. We enjoy creating websites that are both visually appealing and functional, making it easy for users to interact with them.

Why Web Development?

Web development combines technology with creativity, which is what attracts us to it. Through code, we can turn creative ideas into reality while solving practical problems. Additionally, web technologies are constantly evolving, which means we'll always have something new to learn.

Skill Development

  • HTML/CSS - The foundation of building websites. We're learning how to create semantic, accessible web pages.
  • JavaScript - Essential for making websites interactive. We're interested in learning various frameworks and libraries.
  • UI/UX Design - We believe good design is crucial for user experience.
  • Responsive Design - Ensuring websites display well on various devices.
  • Database & Analytics - Learning how to store and analyze user data to provide better services.

Project Features

This Knowledge Quiz project has the following features:

  • Real-time two-player quiz mode, using Socket.io for instant interaction
  • Rich question database covering multiple domains
  • User visit tracking and analysis
  • Historical quiz records and statistics
  • Responsive design, compatible with various devices

Technology Stack

Our project uses the following technologies:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js, Express
  • Real-time Communication: Socket.io
  • Database: SQLite
  • Data Visualization: Chart.js
